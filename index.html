Serverless Devs Serverless 应用全生命周期管理工具

Serverless Devs 是 Serverless 领域的工具链开源项目；通过组件化、应用化的思路，可以让开发者像使用手机一样玩转 Serverless 架构。

开源项目 快速开始
开源建设

Serverless Devs 项目以开源形式进行建设，主张人人可参与，人人可贡献，希望通过开源建设，可以让开发者们"做主"自己的工具，共建"生产力"。

开源 共建 贡献
开放生态

Serverless Registry 秉承着开放生态的思想进行共建，类似于 npm、pypi，Serverless Registry 是 Serverless 领域的生态开放基石。

开放 生态 基石
无厂商锁定

基于组件化和插件化的设计思路，Serverless Devs 是无厂商锁定的开发者工具链/生态，可以帮助开发者快速部署基于多云、多产品的应用。

开放 无锁定 多云

Serverless 应用全生命周期管理

致力于为开发者提供 Serverless 全生命周期管理工具，帮助开发者从体验到开发、部署、运维的路径顺畅，提供好用且感动的 Serverless 工具链

  • 应用纬度的创建更新
  • 应用纬度的环境划分
  • 应用纬度的资源管理

Serverless Registry

Serverless Registry 是一个无厂商锁定的 Serverless 包管理平台，可以帮助开发者非常简单、方便、快速的进行上手 Serverless 架构，助力开发者可以像使用手机一样，玩转 Serverless 架构。

开放生态的基石

Serverless Registry 通过开源开放的形式建设，致力于成为 Serverless 领域的无厂商所定的包管理平台，构建 Serverless 生态基石

类似于 Python 中的 Pypi，Node.js 中的 npm，Serverless Registry 正通过一种更为开发和透明的做法，繁荣 Serverless 生态；

无厂商锁定的做法，有助于开发者可以通过更小的代价，迅速的学习和上手更多 Serverless 架构/平台，或许多云部署，只需一键；

快速体验 Serverless Registry

快速打造现代应用

基于无厂商锁定建设思路，以组件化、插件化的架构基础，Serverless Devs 可以快速帮助开发者构建高可用、高性价比的现代应用

以云原生之名，打造高可用服务

从应用纬度进行定义，通过对资源的组织和编排，通过 Serverless 架构的加持，可以让开发者一键部署高可用、高性价比的云原生应用

从灵活到自由，打造现代应用

以 Serverless 架构为核心，通过组件化、插件化、应用化结合，助力开发者可以快速的实现应用的开发和部署，快速完成全栈应用的上线
